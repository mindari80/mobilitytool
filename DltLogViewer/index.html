좌표 검색

사용 방법
  • SK 좌표: TMAP 내부 좌표계. X는 경도 방향(예: 4571910), Y는 위도 방향(예: 1347000).
  • WGS84: 일반 GPS 좌표. X는 경도(예: 127.0276), Y는 위도(예: 37.4979).
  • 값 입력 후 지도에 표시를 누르면 해당 위치에 마커가 표시됩니다.
  • DLT 로그의 팝업에서 좌표를 복사해 붙여넣어 사용할 수 있습니다.

⚠️ 설치 필수 — CORS Unblock 확장
경로 서버(ntmapstg.tmap.co.kr)는 브라우저의 교차 출처(CORS) 요청을 허용하지 않습니다. 확장 프로그램이 없으면 요청 전송 시 브라우저가 즉시 차단하여 응답을 받을 수 없습니다.
설치 후 브라우저 툴바의 방패 아이콘을 클릭해 ON 상태로 켜주세요.

🚫 CORS 차단 — 요청이 실패했습니다

    

      브라우저가 교차 출처(CORS) 정책에 따라 경로 서버 요청을 차단했습니다.

      CORS Unblock 확장 프로그램을 설치하고 ON 상태로 켜야 요청이 서버에 도달합니다.
    

    

      이미 설치되어 있다면:

      브라우저 툴바의 🛡️ 방패 아이콘을 클릭해 토글을 ON으로 바꾸고 다시 시도하세요.
